== Description ==

This plugin allows users to set their password during registration without requiring email verification.

== Installation ==

1. Upload the plugin folder `direct-password-setup` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
